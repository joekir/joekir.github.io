In SP2010 you can create custom document libraries that integrate with Microsoft Office xml based documents (docx/pptx/xslx etc.) and allow you to reuse SharePoint Office content. For example the "Slide Library" for reusable PowerPoint slides.

Here's what you need to do to access these features:

  • Enable "SharePoint Server Enterprise Site Features" at the site level
  • Enable WebDav support on an end user level, to do this you need to activate the "Desktop Experience" feature on the client OS on which you're using Office 2010

I always forget that the WebDav integration with SharePoint is not a server setting, its a client setting!
